# Portfolio Images

Place your PNG files here:

- **assisme-logo.png** - Project logo (recommended: 64x64px or larger)
- **assisme-screen1.png** - Left phone mockup screen
- **assisme-screen2.png** - Center phone (splash) screen  
- **assisme-screen3.png** - Right phone mockup screen

Or use full URLs in `src/data/portfolio.json` (e.g. `"https://example.com/logo.png"`).
